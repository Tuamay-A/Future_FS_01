# Personal Portfolio - Tuamay Assefa

## Overview
This project is a A modern, responsive personal portfolio website created as part of my first internship task.This project designed to showcase professional skills, projects, and contact information. The portfolio features a clean user interface with dynamic elements and a seamless dark/light mode transition.

## Features
- **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices.
- **Dark/Light Mode**: User-controlled theme switching for better accessibility.
- **Interactive Skills Section**: Dynamic SVG progress circles that animate on category expansion.
- **Scroll Animations**: Smooth entry animations using ScrollReveal.js.
- **Sticky Navigation**: A header that stays at the top and highlights the current section.
- **Dynamic Typing**: Hero section text powered by Typed.js for an engaging intro.
- **Contact Form**: Functional frontend validation for the contact section.

## Technologies Used
- **HTML5**: Semantic structure of the web pages.
- **CSS3**: Custom styling, CSS Variables, Flexbox, Grid, and Media Queries.
- **JavaScript (ES6+)**: Core logic for theme switching, navigation, and skill animations.
- **Boxicons**: High-quality UI icons.
- **ScrollReveal.js**: For scroll-triggered animations.
- **Typed.js**: For the animated typing effect.

## How to Run the Project
1. **Clone the repository**:

2. **Navigate to the directory**:
   ```bash
   cd personal-portfolio
   ```
3. **Open the project**:
   Simply open the `index.html` file in any modern web browser.
   ```bash
   # Or using VS Code Live Server
   code .
   ```
